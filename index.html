Cheatsheet for Common CSS Properties

Common CSS properties
Property Values Description
background-color
  • Color keyword
  • Hexadecimal value
  • RGB value
  • HSL value
 Sets a background color of the selected element(s). background-color: darkblue;
color
  • Color keyword
  • Hexadecimal value
  • RGB value
  • HSL value
 Sets the foreground color, such as text, of the selected element(s). color: red;
border
  • border-width
  • border-style
  • border-color
 The shorthand border property for setting an element's border width, style, and color. border: 4px solid red;
list-style-position
  • inside
  • outside
 Sets the position of a list's marker relative to the content. list-style-position: inside;
display
  • block
  • inline-block
  • none
  • flex
  • grid
 Sets the layout of an element. For example, top to bottom or all in a single line (inline-block). display: inline-block;
font-family
  • First priority font family name
  • Secondary font family names (separated with a comma)
 Selects desired font in a prioritized list. Secondary fonts are used when priority font is unavailable. font-family: Georgia, serif;
font-size
  • Absolute-size keyword
  • Relative-size keyword
  • Pixel size (px)
 Sets the font-size of the element. Note: using pixels is not reccomended because it does not always allow the user to change font-size in their browser. font-size: large;
font-style
  • normal
  • italic
  • oblique
 Sets the style of the font to either normal, italic, or oblique. font-style: italic;
font-weight
  • normal
  • bold
  • lighter
  • bolder
  • 1 - 1000 (inclusive)
 Sets the weight of the font. font-weight: 700;
line-height
  • normal
  • Unitless number * font size
  • percentage
 Sets the distance between lines of text. line-height: 3.5;
